# 💰 Altın & Dolar Portföy Takibi

Altın ve dolar yatırımlarını takip etmek için geliştirilmiş bir portföy yönetim aracı.

## Özellikler

- 📊 Canlı altın ve dolar fiyatları
- 💍 Farklı altın türleri (gram, çeyrek, yarım, tam, cumhuriyet vs.)
- 📈 Otomatik portföy değer hesabı
- 💾 Yerel veri saklama (localStorage)
- 📥 Veri dışa/içe aktar (JSON)
- 📱 Responsive tasarım

## Kullanım

1. Sayfayı açın: [Portfolio](index.html)
2. İstediğiniz varlıkları seçin
3. Sahip olduğunuz adetleri girin
4. "FİYATLARI GÜNCELLE" butonuyla canlı fiyatları çekin
5. Verileriniz otomatik olarak kaydedilir

## Fiyat Kaynakları

- **Dolar Kuru**: Exchangerate-API
- **Altın Fiyatı**: Gold-API
